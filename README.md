---

## Quick Start

```bash
npm install

npm start
```

---

### Available Scripts

In the project directory, you can run:

### `npm start`

Runs a webpack dev server in development mode. Page reloads on edit.

Open [http://localhost:8080](http://localhost:8080)


### `npm run build`

Builds the app for production to the `dist` folder.
